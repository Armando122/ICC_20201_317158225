			
		Practica06
	En esta práctica se utilizó el programa gradle para gestionar mejor la compilación
del código, por lo que para compilar se requiere ubicarse en la terminal en el 
directorio Practica06 y escribir:
		gradle build
Y para ejecutar:
		gradle run

De igual forma se puede compilar de la forma normal ubicandose en el directorio
Practica06/main/java y escribir:
		javac Matrix.java
		javac MatrixTests.java
Y para compilar:
		java Matrix.java
		java MatrixTests.java

El formato de directorios de la practica se encuentra de la forma Practica06/main/java
para utilizar el programa gradle en la ejecución de los archivos de la practica.
